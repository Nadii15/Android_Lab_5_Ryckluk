# Лабораторная работа №5  
**Jetpack Compose: контейнеры, ARGB-модель и Material Design**

## Цель работы  
Повторить основы разработки пользовательского интерфейса на Jetpack Compose, освоить работу с контейнерами `Row` и `Column`, познакомиться с ARGB-моделью цветов, компонентом `Scaffold` и принципами Material Design.

## Краткое описание приложения  
Приложение демонстрирует базовые возможности Jetpack Compose:
- Использование контейнеров `Column` и `Row` для вертикального и горизонтального расположения элементов.
- Отображение текста по символам с применением фильтрации и реверсирования строк.
- Работа с цветами в ARGB-формате: создание непрозрачных и полупрозрачных блоков (`Box`).
- Реализация стандартного экрана с помощью `Scaffold`, включающего `TopAppBar`, основное содержимое и `FloatingActionButton`.
- Разделение кода на отдельные `@Composable`-функции для повышения читаемости и переиспользуемости.
